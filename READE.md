# 🐞 Examen Debug

Proyecto dedicado a la práctica de **depuración de código**.  
Incluye ejercicios donde se corrigen errores lógicos, sintácticos y de ejecución.

## 🔍 Qué contiene
- Código con errores intencionados.
- Soluciones corregidas.
- Explicaciones de los fallos detectados.

## 🎯 Objetivo
Mejorar la capacidad de:
- Identificar errores.
- Comprender trazas de ejecución.
- Depurar de forma eficiente.

## 🛠️ Tecnologías
- Java
